# finapp

App for tracking financial transactions of three different accounts.

## Features

- user account
- add three different accounts (cash,bank,bitcoin)
- remove account
- add transactions for each account
- update transaction
- remove transaction
- pwa
- bitcoin balance converted to fiat (only CZK for now)
